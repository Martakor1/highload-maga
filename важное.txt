Чтобы настроить build в compose файле надо указать build
Чтобы настроить build по кнопке в идее, нужно нажать на docker расширение внизу и запустить хоть раз.
После этого появится run configuration "compose.yaml". После этого её можно будет изменить, добавив туда
--build в принудительном режиме (Always). Тогда будут билдится все те контейнеры, у которых
поменялись Dockerfile или контекст.

Заметка: кажется, gradle работает медленнее чем maven в плане сборки образов.
